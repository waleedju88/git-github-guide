## How to use

Download the guide for your operating system and open it in any web browser (Safari, Chrome, Edge). No internet connection required after downloading.

| File | Operating System |
|------|-----------------|
| [git-github-guide-macos.html](./git-github-guide-macos.html) | macOS (MacBook, iMac) |
| [git-github-guide-windows.html](./git-github-guide-windows.html) | Windows |

---

## What's covered

| Section | Topics |
|---------|--------|
| **Git Setup** | Install Git, configure identity, understand the 3 areas, create first repo, .gitignore |
| **Git Core Operations** | status, add, commit, log, diff, restore — with a live Python calculator demo |
| **Branching & Merging** | Branches, merge, conflict resolution, stash — all on the Python calculator |
| **GitHub Setup** | Account, SSH key authentication, connect local repo to GitHub |
| **GitHub Collaboration Tools** | push, pull, inviting collaborators, Pull Requests, Issues, branch protection |
| **Team Workflow Reference** | Common mistakes & fixes, workflow comparison (Feature Branch / Gitflow / Forking), project checklist |
| **Team Project** | 4-student Task Manager app with roles, issues, PRs, and conflict resolution |
